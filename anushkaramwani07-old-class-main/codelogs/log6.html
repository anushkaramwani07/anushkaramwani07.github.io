What did you look up?

The tag that i looked up was the display CSS property, because different sources explain it in different level of depth. This tag controls how the elements are rendered on a page (block, inline, flex, etc).

Sites Used:

w3school
geeksforgeeks
Mozilla Developer Network
Tech Net

Differences in information:

  1. W3School provides a very simple and user friendly explanation with small examples bit little detail about complex behavior or browser differences.
    • Strengths: Easy to read, examples are straightford and good for beginners
    • Weakness: No in-depth explanation.
  2. Geeks for Geeks have long tutorial style explanations with various examples.
    • Strengths: Variety of good examples that are easy to follow.
    • Weakness: Sometimes the explanation could be vague.
  3. MDN is the most detailed source, with official definitions, browser compatibility tables, and detailed explanation.
    • Strengths: In-depth explanation
    • Weakness: Not reliable for beginners.
  4. TechOntheNet giver short explanation with fewer examples. It not not widely used compared to GeeksForGeeks.
    • Strengths: Focuses on practical use
    • Weakness: Not as many examples

Features that reference page should offer:

  • Links to related properties or tag
  • Readable layout
  • Short definition explaining what that tag or property does.
  • Explain the syntax and what it does

Resource that stood out:

  • Among the four resources, GeeksForGeeks stood out as the most useful. It provides the clearest explanation with multiple examples taht show different situations where teh property is used.
  • The step by step style of this website makes it easier to understand than MDN.
  • For learning and applying the concepts, GeeksForGeeks explains the concepts beginner friendly.
