Code Log 1: Semantic and Non-Semantic Tags

What is the difference between non-semantic and semantic tags in HTML?

  • Non semantic tags don't really tell us anything about the menaing of the context.
  • Example: div tag and span tag
  • Semantic tags describe the meaning and structure of their content.
  • They tell exactly what kind of tag it is, like tell us the content.
  • Example: header tag, article tag, time tag.

What are two (or more) benefits of using semantic tags?

  • Semantic tags describe the meaning and teh structure of that tag, so it is easier to read.
  • It helps with SEO(Search Engines) to understand teh page better.
  • When you come back to the code later, you still know what that tag does cause the purpose of that tag is clear.
