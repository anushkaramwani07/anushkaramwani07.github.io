What do these CSS properties do?

  • The color, background-color, and border-color properties in CSS are used to style elements by changing their appearence.Example color: #FF0000;
  • The color property changes the text color inside an element, such as paragraph, heading, or button.
  • The background-color property sets the color behind the text or content, filling the whole area of that ekement.
  • The border-color property property changes the color that surrounds an element, but it only works if a border is already defined.

How does CSS accept color as a value?

  • Named:Using color names. Example red, blue.
  • RGB:Using Red, Green, Blue values. Example RGB(255, 0, 0).
  • RGBA:with transperency. Example color rgba(255, 0, 0, 0.5)
  • HEXADECIMALUsing hex code. Example color: #FF0000.
  • HSL:Hue, Saturation, Lightness. Example color: hsl(0, 100%, 50%).
