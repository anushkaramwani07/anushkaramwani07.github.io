What do these CSS properties do?

  • COLOR:Changes the text color of an element (like p tag, h1 tag, or div tag)
  • BACKGROUND-COLOR: Sets the background area behind the text or content.
  • BORDER-COLOR: Changes border around that element.

How does CSS accept color as a value?

  • Named:Using color names. Example red, blue.
  • RGB:Using Red, Green, Blue values. Example RGB(255, 0, 0).
  • RGBA:with transperency. Example color rgba(255, 0, 0, 0.5)
  • HEXADECIMALUsing hex code. Example color: #FF0000.
  • HSL:Hue, Saturation, Lightness. Example color: hsl(0, 100%, 50%).
