How does a CSS media query work, and why is it useful?


  • A media query is like giving your website a rules that only apply under certain conditions.
  • Example: If the screen is small, make the text smaller and if the screen is wide, put things side by side
  • It is useful because it lets your website adapt to different devices like phones, tablet and desktops

Media features


  1. Max-width: Means "apply these styles if the screen is smaller than certain width". It is usually used to switch into a mobile layout.
  2. Min- width: Means "apply these styles if the screen is larger than a certain width". It's useful for adding layouts that work better on desktops.
  3. Orientation: This checks if the device is being used in potrait mode or landscape mode
  4. Resolution: Refers to pixel density. You can load sharper images for higher display.
  5. Grid: detects if the output device is grid absed rather tahn pixel based.
